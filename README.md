# baidu_naotu_js
百度脑图3月份即将停止服务,及时下载备份
```
Chrome控制台切到要导出的页面，输入脚本即可
```
或者
```
使用tampermonkey油猴脚本
百度脑图自动导出_tampermonkey.js
打开https://naotu.baidu.com/file/xxxx路径即可自动转换链接导出
```
使用建议
```
xmind文件配合xmind8最佳
.km可通过文本搜索查找关键词
```
